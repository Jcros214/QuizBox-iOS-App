# QuizBox 1.0
An iOS app to control a 'QuizBox'. 

# Features (*italics means still under development*)
* *Directly control QuizBox (Reset, timer, disable quizzers, etc.)*
* Keep Score
  * Correct/Incorrect
  * Quizzing out
    * *Notification of QuizOut*
  * *Team Bonus*
* *Publish results to Quiz/Tournamnet Manager*
